Glossário de <Tags>

Aqui vamos falar um pouco sobre a anatomia do arquivo <html> ou pelo menos o básico para ver algo acontecendo na página.

<!DOCTYPE html>

é uma declaração que define o tipo de documento como HTML

<html></html>

O elemento <html> envolve o conteúdo da página inteira e é conhecido como o "elemento raiz" da página HTML.

<head></head>

O elemento <head> contém o conteúdo não visível da página HTML, como palavras-chave, descrição da página, CSS e declaração de conjunto de caracteres, que são importantes para os resultados de busca e estilização da página.

<meta charset="utf-8">

O elemento <head> contém o conteúdo não visível da página HTML, como palavras-chave, descrição da página, CSS e declaração de conjunto de caracteres, que são importantes para os resultados de busca e estilização da página.

<body></body>

O elemento <body> contém todo o conteúdo visível da página, como texto, imagens, vídeos, áudios e outros elementos.
