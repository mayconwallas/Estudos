Aula 10

aula 10 esta falando sobre eventos DOM os eventos que vamos falar sobre são 

                mouseenter
                mousemove
                mousedown
                mouseup
                click
                mouseout
exemplo na aula: 
                Foi criado uma div e atribui um id e um evendo
                
<div id="area" onmouseenter="entrar()" onmouseout="sair()">
o evento ativa passando o mause.
passando o mouse aqui em cima fica vermelho e ao tirar o 
mouse volta ao normal.    
</div>
                

                mais tambem pode ser colocado como:
                var estilo = window.document.getElementById('area')
                estilo.addEventListener('mouseenter', entrar)
                estilo.addEventListener('mouseout', sair)
                function entrar() {
                     estilo.style.background = 'red'
                }
                function sair() {
                     estilo.style.background = 'aqua'
                }
